#### 주요 기능

##### 직접 정책 제안

- 문제 의식을 갖고 있는 시민이 문제와 관련된 기본적인 정보(해당 지역, 해당 주제 등)와 대략적인 문제의 내용을 작성한다.
- 작성된 내용을 기반으로 AI가 관련 사례 제시와 더불어, 해결책 제시를 위한 가이드 또는 실질 해결책 작성을 위한 골조를 제시한다.
- 이를 기반으로 문제 해결책과 기대효과 등을 사용자가 작성하여 하나의 정책 제안서를 작성할 수 있도록 한다.

##### 커뮤니티 기능

- 다른 사람이 작성한 정책 제안서를 주제별로 정리해서 다른 사용자도 볼 수 있도록 한다.
- 공감 기능, 댓글 달기 기능 등을 통해 정책 제안을 지역 주민들이 공유하고 확인할 수 있도록 한다.
